I removed YouTube Shorts from my browser.

Not because I hate YouTube.

Because I was tired of losing time accidentally.

I would open YouTube for one specific thing:

A tutorial.
A lecture.
A music video.
A bug fix.

Then the Shorts shelf would appear.

And suddenly "just checking one thing" became 10 minutes of scrolling I never meant to do.

So I built NoMoreShorts.

It does one thing:

Remove YouTube Shorts from the interface.

What it blocks:
- Shorts navigation entries
- Shorts shelves in search and browse
- Direct youtube.com/shorts links
- Shorts pages, which redirect back home

What it does not do:
- No tracking
- No analytics
- No servers
- No user data collection

It runs locally in the browser and only touches youtube.com page elements.

This is not published on the Chrome Web Store right now. It is just a personal extension I built to make YouTube less distracting.

The small win:

I can still use YouTube.
But YouTube has fewer chances to use me.
